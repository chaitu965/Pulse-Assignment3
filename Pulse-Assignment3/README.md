# PULSE-ASSIGNMENT3

AI-Driven Documentation Structure Analyzer

## Description
This project analyzes software documentation URLs and extracts
high-level modules and submodules using rule-based logic with optional
AI enhancement.

## Features
- Simple rule-based module detection
- Optional AI inference
- Clean JSON output
- Streamlit UI

## Run
```bash
pip install -r requirements.txt
streamlit run main.py
